# Typography Example

Matches the reference typography examples from `reference/shadcn-ui/apps/v4/registry/new-york-v4/examples`:

- `typography-demo.tsx`
- `typography-h1.tsx`, `typography-h2.tsx`, `typography-h3.tsx`, `typography-h4.tsx`
- `typography-p.tsx`, `typography-lead.tsx`, `typography-large.tsx`, `typography-small.tsx`, `typography-muted.tsx`
- `typography-inline-code.tsx`, `typography-blockquote.tsx`, `typography-list.tsx`, `typography-table.tsx`

## Run

```bash
cargo run --example typography
```
