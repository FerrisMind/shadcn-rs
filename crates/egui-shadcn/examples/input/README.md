Demonstrates the `Input` component with labels, helper text, and form integration.

## Features

- Basic email input
- Disabled email input
- Input with label (Picture)
- Input with button (Subscribe)
- Input with label (Email)
- Input with label and helper text
- Form with username input, helper text, and submit button

## Run

```bash
cargo run --example input --features examples
```
