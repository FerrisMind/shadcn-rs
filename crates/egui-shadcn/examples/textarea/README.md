Demonstrates the `Textarea` component with labels, helper text, and form integration.

## Features

- Basic textarea with placeholder
- Disabled textarea
- Textarea with label ("Your message")
- Textarea with label and helper text
- Textarea with button ("Send message")
- Form with textarea, label, helper text, and submit button

## Run

```bash
cargo run --example textarea --features examples
```
