# Toggle Example

Demonstrates the `Toggle` button component with variants and icon sizes.

## Features

- Toggle with icon and text (Outline variant, Small size)
- Icon-only toggle (Default variant, Small size)
- Icon-only toggle (Default variant, Large size)
- Icon-only toggle (Outline variant, Medium size)
- Toggle with icon and text (Default variant, Medium size)
- Disabled toggle

## Run

```bash
cargo run --example toggle --features examples
```
