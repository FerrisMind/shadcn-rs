Demonstrates the `Dialog` component with modal forms and content.

## Features

- Two buttons to open dialogs ("Open Dialog", "Share")
- Edit profile dialog: title, description, name input, username input, Save/Cancel buttons
- Share link dialog: title, description, read-only input with link, Close button
- Both dialogs: center alignment, scrim overlay, close on background click and Escape key, animations

## Run

```bash
cargo run --example dialog
```
