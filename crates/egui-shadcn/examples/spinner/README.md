# Spinner Example

Demonstrates the loading spinner with Radix Themes leaf animation and shadcn/ui defaults.

## Features

- Size scale matching Radix Themes (`1`, `2`, `3`)
- Loading toggle that overlays the spinner on existing content (Radix `loading` prop behavior)
- Inherits theme foreground color with optional custom accent
- Optional Lucide `loader-circle` rendering mode via `SpinnerVariant::LucideLoaderCircle`

## Run

```bash
cargo run --example spinner
```
