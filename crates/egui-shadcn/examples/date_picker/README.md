# Date Picker Example

Demonstrates shadcn date picker scenarios:

- `date-picker-demo` (single)
- `date-picker-with-range` (range + 2 months)
- `date-picker-with-presets` (select presets + calendar)
- `date-picker-form` (caption dropdown + min/max validation)

## Run

```bash
cargo run --example date_picker
```
