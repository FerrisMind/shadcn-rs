Demonstrates the `Card` component with form layouts and theme switching.

## Features

- Login card with header, email/password inputs, and two buttons (Login, Login with GitHub)
- Project creation card with name input, framework select, and action buttons (Cancel, Deploy)
- Theme toggle switch (light/dark mode) that changes the color palette

## Run

```bash
cargo run --example card --features examples
```
