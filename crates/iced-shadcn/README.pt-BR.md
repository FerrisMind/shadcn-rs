## Visão geral

`iced-shadcn` é planejado como um conjunto de componentes para [iced](https://github.com/iced-rs/iced) inspirados em [shadcn/ui](https://ui.shadcn.com).  
O objetivo é compartilhar a mesma linguagem visual e tokens de tema do restante do ecossistema `shadcn-rs`.

## Status

Este crate está **em desenvolvimento ativo**. A API pública, o modelo de tema e o conjunto de componentes ainda não são estáveis e podem mudar a qualquer momento.

**Coming soon**:

- Catálogo de componentes (quando fizer sentido, em paridade com `egui-shadcn`)
- Guia de temas e tokens
- Exemplos de uso e boas práticas
